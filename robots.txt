# This file is like a little note left on the front door of the
# website for robots (the computer programs search engines like
# Google and Bing send out to go read websites, called "crawlers"
# or "spiders"). It tells them what they're allowed to look at.

# "User-agent: *" means "hey, EVERY robot reading this, this note
# is for you" (not just one specific company's robot).
User-agent: *
# "Allow: /" means "you're welcome to look at everything on this
# site" — there's nothing secret here to hide from search engines,
# so every single page and picture is fair game to read and list.
Allow: /

# This last line points robots to a SECOND helpful note called a
# "sitemap" (see sitemap.xml right next to this file) — it's like
# a table of contents that tells robots exactly which pages exist,
# so they don't have to go hunting for them on their own.
Sitemap: https://sverma2696.github.io/ino_website/sitemap.xml
